### To Do

It is a simple todo list. This application created with ReactJS.

### Features

1.  Create Todo
2.  Search Todo
3.  Table View
4.  List View
5.  Mark Todo as in Progress
6.  Mark Todo as Done
7.  Filter Todo
8.  Select/Unselect Todo
9.  Clear All Todo
10. Reset Todo

### How to use

1.  Download the application from GitHub
2.  Open the application
3.  run from root file

```
npm start
```

4.  Enjoy
